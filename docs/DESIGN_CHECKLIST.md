## ⚠️ UXP CEF 已知限制（落地时必须遵守）

| 限制 | 影响 | 解决方案 |
|------|------|----------|
| 不支持 `flex gap` | 容器内子元素间距 | 用 `margin-left/right/top/bottom` |
| 不支持 `window.confirm()` | 二次确认弹窗 | 用状态导航替代（如 v 删除确认界面） |
| `<button>` 渲染异常 | 点击区域错乱 | `<span role="button" tabIndex={0}>` |
| `<input>` 行为不一致 | 焦点/键盘异常 | 改用 `<sp-textfield>` / `<sp-textarea>` |
| `<input type="checkbox">` 错位 | UI 与 state 不一致 | 用 `<sp-checkbox>` + 整行 click 兜底 |
| `<select>` 不支持 | 下拉样式无法定制 | 改用 `<sp-picker>` + `<sp-menu>` |
| `line-height` 必须带单位 | 行高计算异常 | 统一用 `line-height: 20px` 等 |
| `<form onSubmit>` 不触发 | 表单提交失败 | 用 button onClick 替代 |

## 🎨 视觉规范（已建立的）

### 颜色变量
```css
--color-text           /* 主文字 */
--color-text-secondary /* 次文字（标签、辅助） */
--color-primary        /* 主色 */
--color-bg             /* 背景 */
--color-bg-raised      /* 卡片背景 */
--color-bg-overlay     /* hover 背景 */
--color-border         /* 边框 */
--color-danger         /* 危险/删除 */
--color-warning        /* 警告 */
```

### 尺寸
- **按钮高度**：`padding: 6px 10px` + `border: 1px`，约 34px
- **圆角**：`var(--radius-md)`（统一）
- **字号**：正文 `var(--font-size-md)`、辅助 `var(--font-size-sm)`
- **间距**：基础 `8px`，卡片内 `12px`，区块间 `16px`

## 💡 设计建议

- **间距精确**：8 / 12 / 16px 三档即可，避免 11/13/15 等模糊值
- **配色**：暗色为主，亮色通过 CSS 变量切换
- **图标**：统一使用现有 SVG（`icons/` 目录）
- **错误状态**：用 `var(--color-danger)` 文字 + 圆角描边卡片
- **空状态**：灰色斜体提示文本（见 LayerPicker）
